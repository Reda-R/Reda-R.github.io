R-TYPE PROJECT
README

Welcome to our R-Type project. This is a network video game: It's an emblematic video game of the horizontal scrolling shoot them up. This project is composed of severals parts: Network(Server & Client), CMake (Cross-platform software), Graphical Manager(The game prototype) and the Game Engine(Tools to create games).

BUILD OUR PROJECT UBUNTU/WINDOWS

First Way :

mkdir build -p && cd ./build && cmake .. && cmake --build . --parallel && cd ../

Second Way :

On Ubuntu : ./build.sh
On Windows : ./build.bat

LAUNCH OUR PROJECT

./r-type_server ip port
./r-type_client ip port

If not precise, the machine is set by default to localhost.
